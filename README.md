ArmsFW
============

ArmsFW é uma Lib contendo um conjunto de funcionalidades de extenão que podem ser reutilizadas em qualquer projeto.
Aqui reuni todo o meu conhecimento adquerido ao longo da minha experiencia em programação utilizando a linguagem C#.

## Get started
O primeiro passo para utilizar essa lib é instala-la por meio do pacote nuget `ArmsFW` disponivel no repositorio nuget.org

Para realizar a instalação via NPM, utilize o comando abaixo :

<div>
  <button onclick="navigator.clipboard.writeText(document.getElementById('codeSnippet').innerText)">Copiar</button>
</div>

<pre id="codeSnippet">
Install-Package ArmsFW
</pre> 

Será instalado a versão mais recente do pacote : 

[![NuGet](https://img.shields.io/nuget/v/ArmsFW.svg)](https://www.nuget.org/packages/ArmsFW/)

 ## Funcionalidades

* Abstração da funcionalide de Ler e Escrever configurações.
* Identity Customizavel
* Estrutura deClasses de Repositorio Generico
* Extensões para Tipos de Dados String, Json, Numeros, etc..
